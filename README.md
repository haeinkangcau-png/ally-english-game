Ally(8세)를 위한 영어 단어 플래시카드 게임입니다.
iPad / 갤럭시 탭 브라우저에서 플레이할 수 있어요.

## 매주 단어 업데이트 방법

`words.js` 파일에 새 주차 블록을 추가합니다.

### 1) WORDS_DATA에 새 주차 추가

```javascript
"jun_w1a": {
  label: "6월 1주차 A",
  quizMode: "flash",          // 4지선다 양방향 플래시카드 (생략 시 철자 완성 게임)
  words: [
    { word: "apple",  meaning: "사과",  emoji: "🍎", pron: "ˈæpəl"  },
    { word: "banana", meaning: "바나나", emoji: "🍌", pron: "bəˈnænə" },
    // ... 나머지 단어들
  ]
},
```

> 단어가 많으면 한 주를 `jun_w1a`, `jun_w1b`처럼 나눠도 됩니다.
> 게임은 키 끝의 알파벳(a·b…)을 떼어내 같은 주차 그룹(`jun_w1`)으로 묶어
> "이번 주"로 함께 다룹니다.

### 2) WEEK_ORDER 배열에 새 키 추가

```javascript
const WEEK_ORDER = ["may_w2", "may_w3", "jun_w1"];
//                                       ↑ 새로 추가
```

### 3) GitHub에 words.js 업로드 (덮어쓰기)

저장소 → `words.js` → ✏️ Edit → 붙여넣기 → Commit changes

## 게임 방식

시작 화면에 **4가지 모드** 버튼이 있어요.

| 모드 | 버튼 | 내용 |
|------|------|------|
| 단어 복습 | 📖 이번 주 복습 | 이번 주 단어를 충분히 연습 |
| 단어 퀴즈 | ⭐ 오늘의 퀴즈 | 이번 주 + 지난 단어 섞은 시험 |
| 대화 복습 | 💬 대화 복습 | 표현 듣고 대답 고르기(연습) |
| 대화 퀴즈 | 🗣️ 대화 퀴즈 | 표현 듣고 대답 고르기(시험) |

### 단어 게임 출제 방식 (플래시카드 모드 `quizMode: "flash"`)

객관식 4지선다이며 **한국어→영어 / 영어→한국어** 양방향으로 출제됩니다.
"이번 주"는 같은 주차 그룹의 모든 하위 목록을 합칩니다 (예: `jun_w1a` + `jun_w1b`).

- **오늘의 퀴즈 — 총 12문제**
  - 이번 주: 한→영 3 + 영→한 3
  - 지난 단어(이번 주 제외): 한→영 3 + 영→한 3
- **이번 주 복습 — 최대 20문제**
  - 이번 주(a·b) 단어 전체를 양방향으로 출제
  - 단어 수 N이 10개 이하면 모든 단어를 양방향(2N문제)
  - 10개를 넘으면 방향당 10문제(총 20)로 자르되, **모든 단어가 최소 1번은 나오도록** 균등 분배

> 옛 주차처럼 `quizMode`가 없으면 "이모지 보고 알파벳 버튼으로 단어 완성"하는 철자 게임으로 동작합니다.

### 대화 게임 (대화 복습 / 대화 퀴즈)

표현을 듣고 알맞은 대답을 고르는 4지선다. 각 6문제이며 🔊 발음·🇰🇷 한국어 보기를 지원합니다.

### 별점 (정답률 비례)

문제 수에 상관없이 **정답률**로 매깁니다.

| 정답률 | 별 |
|--------|----|
| 90% 이상 | ⭐⭐⭐ |
| 70% 이상 | ⭐⭐ |
| 40% 이상 | ⭐ |
| 그 미만 | 단어 게임 0개 / 대화·플래시는 최소 1개 |

### 메인 화면 주간 기록

오늘을 포함한 이번 주 7일이 표시되고, 하루에 4종 스탬프 + 별점이 각각 남습니다.
📖 단어복습 · ⭐ 단어퀴즈 · 💬 대화복습 · 🗣️ 대화퀴즈 (같은 모드를 다시 하면 더 높은 별점만 갱신)

## 주차 키 명명 규칙

```
may_w1   → 5월 1주차
may_w2   → 5월 2주차
jun_w1   → 6월 1주차
jun_w1a  → 6월 1주차 (A 묶음)   ┐ 같은 "이번 주"로 묶임
jun_w1b  → 6월 1주차 (B 묶음)   ┘
jun_w2   → 6월 2주차
```
